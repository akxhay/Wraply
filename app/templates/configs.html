Header Transform

Body Transform

Maps call payload fields → outgoing request body. Omit to pass payload as-is, or set {"$PASSTHROUGH": true} explicitly. Use $LITERAL:value for constants.

Response Transform

Maps upstream API response fields → your returned response. Omit to return raw response, or use {"$PASSTHROUGH": true}. Supports JMESPath: items[0].id.
